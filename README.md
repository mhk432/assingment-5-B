**1️⃣ Difference between var, let, and const:**
In JavaScript, **var, let, and const** are used to declare variables. **var** is the older way and it has function scope, which means it is accessible inside the function where it is declared. **let** and **const** were introduced in ES6 and they have block scope, meaning they only work inside the block `{ }` where they are defined. The value of a **var** and **let** variable can be changed later, but **const** cannot be reassigned after it is declared.

**2️⃣ Spread Operator (...):**
The **spread operator (...)** in JavaScript is used to expand or spread elements of an array or properties of an object. It is commonly used to copy arrays or objects, combine multiple arrays, or add new elements easily. It makes code shorter and easier to read.

**3️⃣ Difference between map(), filter(), and forEach():**
In JavaScript arrays, **map(), filter(), and forEach()** are used to loop through elements. **map()** creates and returns a new array after applying a function to each element. **filter()** also returns a new array but only with elements that match a certain condition. **forEach()** simply loops through the array and performs an action for each element, but it does not return a new array.

**4️⃣ Arrow Function:**
An **arrow function** is a shorter and more modern way to write functions in JavaScript. It was introduced in ES6 and uses the `=>` syntax. Arrow functions make the code cleaner and shorter compared to traditional function expressions.

**5️⃣ Template Literals:**
**Template literals** are a feature in JavaScript that allow you to create strings using backticks ( ` ` ) instead of quotes. They make it easier to include variables or expressions inside a string by using `${ }`. This makes string formatting more readable and convenient.
